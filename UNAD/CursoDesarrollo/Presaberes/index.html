Principales Etiquetas HTML

Etiquetas Descripción
<html> </html> Es la etiqueta principal que define el documento tipo web
<body> </body> Comprende todo el contenido que se podrá visualizar en el sitio web
<head> </head> En esta etiqueta podemos indicar titulo del documento, declarar la ruta de las hojas de estilo y de scripts.
<title> </title> Esta etiqueta se usa dentro de la etiqueta HEAD y sirve para declarar el nombre del sitio que aparecerá en la barra de título del navegador web.
<br> Con esta etiqueta podemos hacer un salto de línea. Esta etiqueta en particular no tiene etiqueta de cierre
<p> </p> Esta etiqueta nos permite escribir párrafos y al cerrarla hace un doble salto de línea.
<h1> </h1>, <h2> </h2>, <h3> </h3>... Estas etiquetas nos permiten de forma automática indicar el tamaño del texto que encerremos en ella. La etiqueta H1 indica el texto más grande y va disminuyendo hasta H6.
<table> </table> Con esta etiqueta podemos crear una tabla dentro del documento web. Es necesario combinarla con las etiquetas TR y TD para indicar filas y celdas respectivamente.
<div> </div> Es una etiqueta que hace parte del lenguaje semántico de HTML y sirve como una caja contenedora de diferentes elementos que guardan cierta relación entre ellos. De esta manera a través de los estilos CSS se pueden afectar de manera general sin necesidad de editarlos de manera individual.
<a href> </a> Esta importante etiqueta le da sentido a la nevegación web ya que gracias a esta se pueden crear los diferentes enlaces que tiene un sitio web.
<div> </div> Es una etiqueta de uso extendido como contenedor de cualquier clase de contenido. No aplica el concepto de semántica en la web
<article> </article> Es una etiqueta semántica que contiene información con un tema en concreto
<section> </section> Otra etiqueta semántica que encierra mayor contenido. Por ejemplo, puede encerrar varias etiquetas <article> con información en común

Estilos de CSS

Propiedades Descripción
Colors Casi cualquier clase de creemos en este lenguaje tiene el atributo color con el cual se puede modificar. Por ejemplo, para la etiqueta <p> </p> se puede indicar el color del texto de la siguiente forma: 
	p{
            color:blue;
            }
Backgrounds Para los fondos de los elementos también se puede aplicar estilos como color o incluso agregar una imagen. La etiqueta <body> también tiene el atributo background con lo cual se podría aplicar estilos al fondo general del documento web. Veamos cómo se carga una imagen a los párrafos dentro de la etiqueta <p>: 
	p{
            background-image: url(fondo.jpg);  
            }
Borders Los bordes dentro de una tabla también se pueden editar gracias al atributo border y sus diferentes opciones como border-radius, border-color, border-width, entre varios.
	td{
                border-radius: 16px;
                border-color: red;
                border-style: outset;
                border-width: 5px;
            }
Font Siendo el texto el contenido web por excelencia, era obligatorio contar con un atributo que permitiera aplicar estilos tanto de forma, tamaño, color y un largo etcétera. Por ello tenemos el atributo Font con el cual podemos aplicar diferentes estilos al texto. Por ejemplom, aplicamos estilo cursivo, negrilla, tamaño de 18 px y tipo de fuente cursive al texto de este párrafo. 
	td{
            font-size: 18px;
            font-family: cursive;
            font-style: italic, bold;
            }
Position
Otro atributo usado con bastante regularidad es Position ya que con él podemos editar la posición de cualquier elemento dentro del sitio web. Por ejemplo, un texto, un bloque de texto o una imagen. Las propiedades que se pueden aplicar son static, relative, fixed, absolute, sticky. Por ejemplo, aplicamos el estilo absolute a este párrafo y vemos como adquiere la posición de la fila determinada por la etiqueta <tr> ignorando los parámetros de su contenedor <td> saliéndose y ocupando el espacio de la fila inferior. Esta no es la práctica correcta para el manejo de información dentro de las tablas, pero como ejemplo, es una forma fácil de ver la función de la posición absoluta. 
	div.absolute{
                position: absolute;
                }
margins margin es una propiedad muy importante que afecta el espacio alrededor de un elemento en el documento. Se pueden editar las márgenes superior, inferior, izquieda y derecha como se muestra en la imagen:
width y height Estas propiedades se pueden aplicar a casi cualquier elemento en el DOM que tengan un tamaño. Width por su traducción comprende el ancho del objeto y Height, su alto. Estos tamaños se pueden definir con medidas absolutas como pixeles, centímetros y pulgadas; o medidas relativas como porcentajes vh o vw.
